# Managed Runtime Environments

Managed runtime environments let you run durable workflows without managing infrastructure. Focus on your code while Runtara handles deployment, scaling, and operations.

## Overview

Runtara's managed runtime environments provide:

- **Zero-config deployment** - Deploy your workflows with a single command
- **Automatic scaling** - Scale up and down based on demand
- **Built-in observability** - Monitoring, logging, and tracing out of the box
- **Global distribution** - Run close to your users with edge deployment

## Deployment Options

### Runtara Cloud

The fully managed option. Deploy your workflows and let Runtara handle everything else.

### Self-Hosted Managed

Run the Runtara runtime on your own infrastructure with managed operations through the Runtara control plane.

### Hybrid

Combine cloud and self-hosted deployments for flexibility and compliance.

## Key Features

### Native Binary Execution

Your workflows compile to native binaries, ensuring fast startup times and minimal resource usage.

### Secure Isolation

Each execution runs in an isolated environment with strict resource limits and security boundaries.

### Automatic Updates

Runtime environments are automatically updated with security patches and performance improvements.
